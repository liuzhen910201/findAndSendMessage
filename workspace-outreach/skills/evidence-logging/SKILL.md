# Purpose

Create an auditable trail for each company outreach step.

# Required Logs

- input row snapshot
- official website evidence
- inquiry page URL
- drafted message
- approval decision
- send result
- timestamp

# Recommended Evidence

- page title
- domain
- screenshot path if available
- result code
- operator or run id

# Output

- evidence_path
- checked_at
- sent_at
- status_history
